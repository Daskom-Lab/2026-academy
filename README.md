# 2026 Academy

Welcome to the **Academy 2026** repository!

This repository serves as the central place for all academy materials, assignments, documentation, and project archives throughout the program. We hope everyone can learn, collaborate, and grow together into better developers.

> **Build. Learn. Collaborate.**

---

## 📖 About

Academy is an intensive bootcamp organized by DASKOM to prepare future software engineers through structured learning, hands-on projects, and mentorship.

Throughout the academy, participants will explore various aspects of modern web development while working together to solve real-world problems.
